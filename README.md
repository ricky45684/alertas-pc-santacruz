## Cómo usar

1. Abrí la URL desde cualquier dispositivo
2. Seleccioná tipo de alerta y nivel
3. Completá fecha, períodos (M/T/N) y zonas afectadas
4. Agregá resumen del evento y hora de emisión
5. Descargá la imagen PNG o copiala al portapapeles

---

## Instalar como app (opcional)

En Chrome (Android o escritorio):
- Abrí la URL
- Tocá el ícono de instalar en la barra de direcciones
- La app queda en el inicio del dispositivo

En Safari (iPhone):
- Abrí la URL
- Tocá "Compartir" → "Agregar a pantalla de inicio"
